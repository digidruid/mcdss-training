Inspect Elements

Elements

Debug the generated html code and styling. In this example the style.css has the following. 
.code {
    background-color: black;
    color: lightgray;
    padding: 10px, 10px;
}
The expexted result would have been.
Most browsers have an Inspect or Developer Tools mode.
Looking at the element we see the font colour resolving to red.
Drilling down on the Styles view on the right, we can see that we are not picking up the style from style.css.
This if because the <code>Inspect</code> style is not defined for the element, but for the class ie. '.code'. We have 2 options to fix this.
  1. Add a 'code' style in style.css
  2. Add class="code" in the html
I opted for the second options as the will give more control on inline styling.
