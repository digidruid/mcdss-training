Basic File and Text Manipulation

Editors

  • nano: Basic editor available in most distributions
  • vim: 'Expert Friendly' editor with a lot of extendability
For vim just remembet Esc, :wq, Enter

Basic Commands

  • Copying, moving, deleting files: cp, mv, rm
  • Viewing and editing files: cat, less, more, grep
  • Comparing files: diff
  • Symbolic and hard links: ln

Important Folders

  • /bin - most of the scripts and binary files in the system
  • /home - all user files
  • /dev - devices
  • /etc - configurations
  • /lib - libraries
  • /proc - system runtime details
  • /var - variable data (Logs)

Commands

  • ls: Listing
  • cd: Change directory
  • pwd: Print working directory
  • mkdir: Make directory
  • rmdir: Remove directory
  • man: Man pages

Permissions

  • chmod: Change permissions
  • chown: Change owner
  • chgrp: Change group
  • groups: List groups
  • sudo: Super User Do
