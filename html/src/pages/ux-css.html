What is CSS?

CSS stands for Cascading Style Sheet.

In the last section of the Styling demo, we moved the styles into the <head><style> section. We can move the content into a .css file. This allows styling to be defined separate from layout.

Linking

CSS files are linked into html files with the following syntax. 
    <link rel="stylesheet" type="text/css" href="style.css">

Selectors

CSS styles have the following syntax. 

        <selector> {
            <property> : <value> ;
            <property> : <value> ;
            ..
        }

Selectors tell the styling engine where to apply the styles. There are 4 selector types.

  • Element
    • Example: h4 { color: blue; }
    • Matches HTML tags like h1, div, p etc
  • Classes
    • Example: .warning { color: red; }
    • Matches tags like <div class="warning" ...
  • IDs
    • Example: #item1 { color: green; }
    • Matches only one tag like <button id="item1" ...
  • All
    • Example: * { color: yellow; }
    • Matches everything. This should not be used unless there is a very good reason. It tends to break more things than it fixes.
