Style Cascading

Styles are applied in a cascading / inheritted way between elements. As you can see all the fonts of all the elements have update to Arial. This is due to the style="font-family: Arial" on the <body> element


If your web application has a lot of styling it can become very tedious and error prone to update individual elements. Luckiy styles can be grouped into classes and also be applied based on element type. For the previous example, we missed styling the Day 1: Web Applications header. We are going to fix this now with a generic style on the <h2> element.

We are also going to make a class for completed items. If we decide to change the style, we can do it in one place.
