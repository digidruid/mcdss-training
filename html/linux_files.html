Users and the Filesystem

Filesystem Layout

  • One structure regardless of physical disks
  • Root folder /
  • Device and Network Shares mounted into the folder structure

The root folder


What do we see?

  • File Types
    • -: 'Normal' File
    • d: Directory
    • l: Link
  • Permissions for User, Group and Other
    • r: read
    • w: write
    • x: execute
  • User / Group Ownership
  • Size
  • Last Change Time
  • Filename
    • Files starting with '.' are "hidden" and usual has a special purpose

Important Folders

  • /bin - most of the scripts and binary files in the system
  • /home - all user files
  • /dev - devices
  • /etc - configurations
  • /lib - libraries
  • /proc - system runtime details
  • /var - variable data (Logs)

Commands

  • ls: Listing
  • cd: Change directory
  • pwd: Print working directory
  • mkdir: Make directory
  • rmdir: Remove directory
  • man: Man pages

Permissions

  • chmod: Change permissions
  • chown: Change owner
  • chgrp: Change group
  • groups: List groups
  • sudo: Super User Do
