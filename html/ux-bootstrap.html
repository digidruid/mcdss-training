Bootstrap

What is Bootstrap?

Bootstrap is a web page / application layout framework that utilizes CSS and JavaScript. It enables you the build reactive web app layouts which will scale to mobile page views.

The Box Model

Before layout frameworks, you had to use margins, borders and padding on a div to layout your web pages. The sizes of these style attributes can be specified in a number of different units, like percentage, pixels, em etc. There was quite a bit of planning on paper involved and to make things worse, not all broswers handled the borders and padding the same way.

You can use the Browser Inspect mode to look at an elements' box parameters

12 Column Grid Overview

Bootstrap uses the concept of the screen being devided into 12 columns. The columns are layed out in a 'container' with 'row's and the 'col's.

Code:

<head>
    <style>
    .box {
        background-color: rgb(22, 141, 68);
        color: lightgray;
        margin: 5px;
    }
    </style>
  ...
</head>
<body>
<div class="container">
    <div class="row">
        <div class="col">
            <h1>Welcome to The Box Model</h1>
        </div>
    </div>
    <div class="row">
    
        <div class="col box">
            <h2>Hello World</h2>
        </div>
        <div class="col box">
            <h2>Have a good day !!!</h2>
        </div>
    </div>
</div>
...
</body>

Result

  1. Column Widths:
    You can specify how many columns an element should span using classes like
    • col-
    • col-sm-
    • col-md-
    • col-lg-
    • col-xl-
    For example, col-md-6 means the element will take up 6 out of 12 columns on medium-sized screens and above.
  2. Responsive Behavior: The grid system adapts based on screen size:
    • col-: Applies for extra-small screens (less than 576px).
    • col-sm-: Applies for small screens (≥576px).
    • col-md-: Applies for medium screens (≥768px).
    • col-lg-: Applies for large screens (≥992px).
    • col-xl-: Applies for extra-large screens (≥1200px).
  3. Auto Layout: You can use col without specifying a number, allowing the column to automatically adjust its width based on the number of sibling columns.
